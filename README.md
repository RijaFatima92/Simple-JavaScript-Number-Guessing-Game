# Simple-JavaScript-Number-Guessing-Game

Description:
A lightweight, interactive web-based game where players have 3 attempts to guess a randomly generated number between 1 and 10.

🛠️ Features:
Dynamic Feedback: Real-time messages for "Too High" or "Too Low."

Input Validation: Prevents invalid entries and out-of-range numbers.

Interactive UI: Clean card-based design with hover effects and responsive buttons.

Game Logic: Uses Math.random() for target generation and state tracking for attempts.

🚀 Tech Stack:
HTML5 (Structure)

CSS3 (Flexbox, Card UI)

JavaScript (ES6 Logic)
